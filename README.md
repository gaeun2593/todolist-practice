## JAVA : ToDoList-Practice
자바 콘솔창을 이용해 간단한 투두리스트 구현한 개인 프로젝트

## 📍주요 기능
### 유저
1. 회원 가입 (CREATE)
2. 로그인 (READ)
3. 닉네임 변경 (UPDATE)
4. 회원 탈퇴 (DELETE)

### 카테고리
1. 카테고리 전체 조회 (READ)
2. 카테고리 추가

### 투두리스트
1. 투두리스트 전체 조회
2. 유저별 투두리스트 조회
3. 카테고리별 투두리스트 조회
4. 투두리스트 등록
5. 투두리스트 상태 변경 (완료 체크)
6. 투두리스트 삭제
